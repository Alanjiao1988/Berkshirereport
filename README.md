## 阅读指南

每家公司的文件夹下按日期存放该公司的历次研究报告（同一公司可能有多个时间点的更新版）。目录内的 `README.md` 是该公司的阅读入口，会标明最新版、历史版本和核心差异。多数单公司深度报告使用以下结构：

1. AI研究偏见自觉（信息丰富度评级 + 局限性声明）
2. 关键数据验证记录
3. 生意本质分析（段永平）
4. 护城河评估（巴菲特）
5. 逆向思考与风险清单（芒格）
6. 管理层评估（段永平+巴菲特）
7. 行业与文明趋势（李录）
8. 估值与安全边际
9. 综合决策备忘录（含具体买入/加仓/卖出价格区间）

少数目录包含团队版报告，用于从业务、财务、行业、风险等多视角交叉审视同一家公司，可与单篇深度报告互相校验。

**赶时间只看结论**：直接跳到每份报告末尾的"第八步：综合决策备忘录"，包含汇总表格、买卖信号和具体价格区间。

## 数据审计

所有报告发布前均经过程序化数据抽检：从报告中随机抽取15%的数据点，逐一与 macrotrends / stockanalysis 等独立信源比对，偏差超过1%的项目会被打回修正，通过后方可发布。审计记录见各报告文件末尾或同目录下的审计说明。
